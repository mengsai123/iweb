2.使用浮动布局时常见的问题
若容器中的子元素浮动起来，则不再占用布局空间
如果容器内所有的子元素全部浮动，则父容器不再有高度
解决方案：给父容器添加新的没有浮动的子元素,
.col {
	float:left;
}
.row:after {				/*在父容器尾部生成新元素*/
	content: '';
	display: block;
	clear: both;			/*清除行浮动造成的影响*/
}
